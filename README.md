# Toast — P2P real-money gaming: investment research brief

A self-contained static research site (zero external requests). Internal work product; **not legal or investment advice.** All statistics are source-linked; claims are tiered A (primary) / B (secondary) / C (per company/call) / D (our estimate).

The deployed page is set to `noindex, nofollow`. Serve locally with `python3 -m http.server` from this directory.
